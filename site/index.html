Psilon.ai

Run any model on any machine.

Psilon democratizes AI access for consumers. It removes the NVIDIA ecosystem dependency from AI by porting PyTorch to open, standardized GPU APIs — Vulkan, DirectX 12, and WebGPU — and it makes frontier-scale models runnable, unmodified, on ordinary consumer gaming hardware.

Vulkan DirectX 12 WebGPU CUDA-compatible

Why Psilon exists

Today, serious frontier AI models run on one vendor's hardware, through one vendor's software stack. If your GPU doesn't speak CUDA — or your model doesn't fit in your VRAM — you're locked out. Psilon enables everyone to participate in the singularity.

Open the hardware

A highly optimized PyTorch backend for Vulkan, DX12, and WebGPU that aims to match or exceed CUDA performance on the same silicon. AMD, Intel, NVIDIA, Apple, and the browser — one model, every GPU.

Break the VRAM ceiling

General-purpose AI model streaming: frontier models run in their unmodified entirety on low-cost hardware by streaming layers through the GPU instead of holding the whole model in VRAM.

First release · Pillar 1

psiTorch — PyTorch on open GPU standards

psiTorch is a highly optimized PyTorch library and backend for Vulkan, DirectX 12, and WebGPU. It keeps the PyTorch name and API surface, so existing models and code run unchanged — torch.device("vulkan") works the way you'd expect.

The Vulkan port is often as fast or faster than CUDA on the same hardware — and it is positioned to pull further ahead as new drivers expose more tensor-core primitives to Vulkan and DX12.

Full operator surface

2,691 ATen operator registrations, with matrix-core acceleration (tensor cores, AMD WMMA, Intel XMX) and automatic hardware tier detection.

One shader source

A single shader codebase compiles to SPIR-V for Vulkan, DXIL for DirectX 12, and WGSL for WebGPU.

Training included

Not inference-only. psiTorch targets the full PyTorch experience — training and inference — on open APIs.

First release · Pillar 2

psiCache — stream frontier models on consumer hardware

Streaming an AI model does not require quantizing it, pruning it, or otherwise cutting it down to fit in limited VRAM. It is a new technique for processing frontier models in their unmodified entirety on modest consumer hardware. They may not be fast — but they work, today, uncompromised.

How it works

  1. Compile-time profiling. psiCache sits in the PyTorch compiler. During compilation it captures metadata about the model's structure and access patterns and builds a proactive caching profile.
  2. Adaptive execution. When the model runs on arbitrary hardware, psiCache uses that profile to optimize itself for the machine it lands on — orchestrating VRAM, CPU RAM, and NVMe as a unified cache hierarchy.
  3. Layer-by-layer streaming. Arbitrarily large models stream through the GPU one layer at a time, with prefetch running ahead of compute. The entire model never needs to be resident in VRAM at once.

Streaming and quantization are complementary, not competing. Combine psiCache streaming with familiar quantization techniques to raise tokens-per-second — or run the full-precision original when fidelity matters. The point is that the choice is finally yours.

First release · Pillar 3

Fused Hugging Face libraries

Psilon ships a family of the top Hugging Face AI Python libraries — fused and optimized to perform better on CUDA, Vulkan, DX12, and WebGPU alike. High-overhead Python paths in the standard libraries are replaced with fused native GPU implementations, so the ecosystem you already use gets faster on every backend, including CUDA.

Drop-in ecosystem

Keep your transformers-style workflows. The fused libraries preserve the familiar APIs while eliminating interpreter overhead on the hot path.

Every backend wins

Fusion benefits aren't limited to the new backends — the same optimizations accelerate CUDA too. There is no penalty for coming along.

AI as a first-class streamable media type

By transforming AI into a streamable consumer media type, Psilon enables everyone to participate in the singularity. AI is no longer a monopoly for a few big companies to define or control.
